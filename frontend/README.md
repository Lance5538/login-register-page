# frontend

This folder is now the primary implementation of the login, register, and dashboard flow.

## Run locally

```bash
npm install
npm run dev
```

## Build for preview

```bash
npm run build
```

The Vite `base` setting uses relative asset paths so the built output is easier to preview locally.

## Project note

- `frontend` is the main version to continue developing.
- The repository name is `smart-warehouse-system`.
